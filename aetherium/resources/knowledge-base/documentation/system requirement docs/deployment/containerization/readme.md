### Basic Commands

```bash
# Build an image
docker build -t myapp:latest .

# Run a container
docker run -p 3000:3000 myapp:latest

# List containers
docker ps -a

# View logs
docker logs <container_id>

# Execute command in running container
docker exec -it <container_id> /bin/bash
